## Detalhes

Trata-se de um sistema gerador de classes Gets e Setters junto com Fachadas de comunicação com os métodos e com banco de dados.

Funciona indicando o host, usuário, senha e banco de dados que se deseja gerar as classes, após isso deve-se escolher para quais tabelas as classes devem ser geradas. Ao final, as classes são geradas na pasta classes_geradas de onde se deve recortá-las e colar na pasta classes do projeto que está sendo desenvolvido.

Existe uma pasta auxiliar na pasta classes_geradas chamada segurança, a qual contém diversas classes auxiliares para conexão com banco de dados e outros.
